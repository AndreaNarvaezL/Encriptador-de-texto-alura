# Text Encryption Tool (Learning Project)

Aplicación desarrollada como ejercicio práctico para comprender la transformación de texto mediante reglas definidas.

## Descripción
Este proyecto permite encriptar y desencriptar texto utilizando un conjunto de reglas básicas.

## Tecnologías
- HTML
- CSS
- JavaScript

## Funcionalidad
- Encriptación de texto
- Desencriptación de texto

## Security Perspective
Aunque este proyecto utiliza una lógica básica de transformación de texto, permite introducir conceptos relevantes como:
- Importancia de la protección de la información
- Diferencia entre ofuscación y criptografía real
- Riesgos de implementar mecanismos de seguridad no robustos

## Consideraciones
Este tipo de implementación no debe utilizarse en entornos productivos, ya que no cumple con estándares de seguridad criptográfica.

## Aprendizajes
Este ejercicio permitió comprender conceptos iniciales relacionados con protección de datos y la importancia de implementar mecanismos de seguridad adecuados.
